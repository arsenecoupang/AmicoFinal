# 🧪 AMICO 앱 전체 기능 테스트 가이드

## 🎯 테스트 준비 완료!

### ✅ 현재 테스트 설정 상태

- **MVP 투표**: 시간 제한 해제 ✅ (언제든 투표 가능)
- **날짜 제한**: 해제 ✅ (모든 날짜의 채팅방에서 투표 가능)
- **기존 사용자**: user1, user2, user3, admin, admin01 등 이미 존재

## 📋 전체 기능 테스트 순서

### 1. 🔑 **로그인 테스트**

```
브라우저에서 localhost:3000 접속
→ 기존 계정으로 로그인: user1, user2, user3 중 아무거나
→ 또는 새 계정 생성
```

### 2. 🏠 **메인 화면 테스트**

```
로그인 후 메인 화면에서 확인:
- 온도 표시 ✅
- 온도 랭킹 ✅
- MVP 투표 버튼 (테스트 모드 표시) ✅
- 퀴즈 시작 버튼 ✅
```

### 3. ❓ **퀴즈 기능 테스트**

```
메인 화면에서 "퀴즈 시작" 클릭
→ 퀴즈 선택 (밸런스 게임)
→ 선택지 클릭하여 채팅방 입장
```

### 4. 💬 **채팅 기능 테스트**

```
퀴즈 선택 후 채팅방 입장
→ 메시지 입력 및 전송
→ 실시간 채팅 확인
→ 다른 브라우저/시크릿모드로 다른 계정 접속하여 같은 방 입장
→ 실시간 채팅 동작 확인
```

### 5. 🏆 **MVP 투표 테스트**

```
메인 화면에서 "MVP 투표하기" 클릭
→ 참여한 채팅방 멤버들 목록 확인
→ MVP 후보 선택 후 투표
→ 투표 현황 확인
→ "결과 확정" 버튼으로 MVP 선정 (관리자 기능)
→ 선정된 MVP의 온도 10도 증가 확인
```

### 6. 🌡️ **온도 시스템 테스트**

```
메인 화면에서 온도 랭킹 확인
→ MVP 투표 후 온도 변화 확인
→ 프로필 정보 확인
```

## 🔧 멀티 사용자 테스트 방법

### **방법 1: 브라우저 여러 개**

```
1. Chrome: user1로 로그인
2. Edge/Firefox: user2로 로그인
3. 시크릿 모드: user3로 로그인
```

### **방법 2: 친구들과 함께**

```
1. 친구들에게 localhost:3000 링크 공유
2. 각자 다른 계정으로 로그인
3. 같은 퀴즈 선택하여 채팅방 입장
4. 채팅 후 MVP 투표 진행
```

## 🎮 테스트 시나리오 예시

### **시나리오 1: 기본 플로우**

```
1. user1로 로그인
2. 퀴즈 "한식 vs 양식" 선택 → "한식" 클릭
3. 채팅방에서 메시지 입력
4. user2로 다른 브라우저에서 같은 퀴즈 → "한식" 선택
5. 두 명이 채팅 진행
6. MVP 투표에서 서로 투표
7. 결과 확정하여 MVP 선정
```

### **시나리오 2: 풀 기능 테스트**

```
1. 3-4명이 여러 채팅방 참여
2. 각 방에서 활발한 채팅
3. MVP 투표에서 모든 참여자가 후보로 나타나는지 확인
4. 투표 진행 및 집계 확인
5. MVP 온도 증가 확인
6. 온도 랭킹 업데이트 확인
```
